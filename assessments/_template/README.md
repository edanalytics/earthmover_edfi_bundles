This is a template for an earthmover assessment bundle. It is mostly empty, with a few example values to demonstrate list syntax and some common logic we use in assessment templates.

Please review other bundles for real examples:
- [BASC3-BESS](https://github.com/edanalytics/earthmover_edfi_bundles/tree/main/assessments/BASC3_BESS)
- [CogAT](https://github.com/edanalytics/earthmover_edfi_bundles/tree/main/assessments/CogAT)
